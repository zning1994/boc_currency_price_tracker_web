# BOC currency price archive

Raw JSON snapshots of Bank of China exchange rates, updated every 30
minutes. This is the data CDN — for the public site with charts and an
API reference, go to <https://bocurrencyprice.techina.science>.

Direct file URL pattern:

```
https://data-bocurrencyprice.techina.science/BOC_CURRENCY_PRICE/<CCY>/<YYYYMMDD>.json
```

For example: <https://data-bocurrencyprice.techina.science/BOC_CURRENCY_PRICE/USD/20260425.json>
